# MIRA — Market Intelligence Research Analyst

Scaffold for a premium hedge-fund style portfolio website for Chris Wall.

Installed:
- Next.js (App Router)
- Tailwind CSS
- Framer Motion
- Chart.js (react-chartjs-2)

Run:
- npm install
- npm run dev

Notes:
- Add `npx shadcn-ui init` and add shadcn components as needed.  
- Replace placeholder data and wire to API routes for live data.
